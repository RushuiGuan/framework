Method AddConfig

AddConfig<ConfigType>(IServiceCollection, bool)

Registration for the Configuration Class C. C has to have the base class of ConfigBase and it also requires a constructor with a single parameter of type IConfiguration The call will run the validate method right after the object creation. It is critical to validate configuration data. A misspelled property name in the appsettings.json file could lead to null config values.
