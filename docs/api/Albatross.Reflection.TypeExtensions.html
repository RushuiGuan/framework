TryGetClosedGenericType(Type, Type, out Type?)

Provide with a class type and a generic type\interface definition, this methods will return true if the class derives\implements the generic type\interface. It will also output the generic type.

Example: public class Test: IEnumerable<int> { }

var result = typeof(Test).TryGetClosedGenericTypes(typeof(IEnumerable<>), out Type type); Assert.True(result); Assert.AreSame(typeof(IEnumerable<int>), type);
