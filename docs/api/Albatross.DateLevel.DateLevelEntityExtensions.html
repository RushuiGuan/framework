GetOverlappedDateLevelEntities<T>(IEnumerable<T>, DateOnly, DateOnly?)

The method will find the date level entries in source that overlap with the given date range. If the end date is not specified, the method will find the next record in the series and set the end date to the day before its start date. If the next record does not exist, the end date will be set to the max end date. This method does not require a key, it assumes the supplied source only contains a single date level series.

UpdateDateLevel<T>(ICollection<T>, Action<T>, DateOnly, DateOnly?, bool)

This function will attempt to update date level values between start and endDate. If the endDate is not specified, the function will find the next record in the series and set the end date to the day before its start date. If the next record does not exist, the end date will be set to the max end date. The function will update only. It will not insert records if no existing records that overlap the specified date range are found.
