Merge<Src, Dst, TKey>(IEnumerable<Dst>, IEnumerable<Src>, Func<Src, TKey>, Func<Dst, TKey>, Action<Src, Dst>?, Action<Src>?, Action<Dst>?)

Perform a full outer join between 2 collections by comparing the key generated by the specified func delegates. The method also accepts delegate parameters that handles the join, left join and right join use cases. The method doesn't consider the key value of default(TKey) a valid key value. Therefore any source item with the key value of default(TKey) will be handled by the notMatchedByDst delegate.
