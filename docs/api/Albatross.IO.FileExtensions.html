Prepend(Stream, Stream, bool)

insert the content of the source stream to the destination stream. If the copySource is true, the source stream is copied to a temporary file before the prepend operation and therefore is untouched. Otherwise the source stream is used as the buffer file. The content of the source stream is not changed, however the last modified date of the source stream would be updated. Note that the positions of the source stream and the destination stream do not need to be at the beginning. The resulting file will have the content of the source stream starting from its current position and the content of the destination stream starting from its current position.
