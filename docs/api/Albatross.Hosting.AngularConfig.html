RequestPath

Use by the hosting aspnetcore web app. Default is String.Empty. This property should reflect the relative path for the angular app to the aspnetcore web app. It should start with a / but never end with one. For example, if the angular path is: http://localhost/demo/ui and the aspnetcore web app path is: http://localhost/demo, the request path should be /ui
