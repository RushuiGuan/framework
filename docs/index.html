Albatross Framework

Albatross is a .Net development framework that allow developers to quickly bootstrap an application with cross cutting functionaties such as logging, caching and config management. It builds intergration points to other libraries so that developers can work with a consistant platform without having to solve the same technical problems repeatly.

Here is a list of technogies adopted by Albatross
