GetEndPoint(IConfiguration, string, bool)

For C# the HttpClient class will remove any relative path if the BaseUrl does not end with a slash. For example: http://localhost/beezy will becomes http://localhost unless base url is set as http://localhost/beezy/ For the request url, if it starts with a slash, it will be considered as a root url. By default, we shouldn't use any slash in the request url. The call will append Slack '/' to the endpoint by default if it doesn't already end with it. If this behavior is not desired, set ensureTrailingSlash to false
